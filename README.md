# E-commerce Frontend M2

Esto es un proyecto de una tienda en linea  hecho con HTML, CSS, Bootstrap y JavaScript

## Contiene 
-Catalogo con productos dinamicos 
-Página de productos con detalles obtenidos dinamicamente  atraves del id 
-Carro de compras con un contador en el nav
-Persistencia en localStorage para que se guardn los productos en el carro
-Página  principal con enlaces a las secciones y desde el nav puedes dirigirte tanto a las secciones como a las demas paginas
-Pagina de Contacto
-Página de carro donde te muestra los productos seleccionados para su compra y donde los puedes eliminar.
-Extructurado con  assets que contiene las carpetas de css, img y js y en la raiz del proyecto se encuentrasn sus paginas html.
